**题目：**循环输出列表

**程序分析：**无。

程序源代码：

实例
--
```
#!/usr/bin/python # \-\*\- coding: UTF-8 \-\*\- if  \_\_name\_\_ == '\_\_main\_\_': s = \["man","woman","girl","boy","sister"\]  for  i  in  range(len(s)): print  s\[i\]
```
以上实例输出结果为：
```
man
woman
girl
boy
sister
```

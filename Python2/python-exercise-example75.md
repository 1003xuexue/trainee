**题目：**放松一下，算一道简单的题目。

**程序分析：**无。

实例
--
```
#!/usr/bin/python  \# -*- coding: UTF-8 -*-  if  \_\_name\_\_ == '\_\_main\_\_': for  i  in  range(5): n = 0  if  i != 1: n += 1  if  i == 3: n += 1  if  i == 4: n += 1  if  i != 4: n += 1  if  n == 3: print  64 \+ i
```
以上实例输出结果为：
```
67
```
